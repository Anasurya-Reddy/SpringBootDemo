# 🚀 Spring Boot Demo Project

This is a simple **Spring Boot web application** demonstrating basic functionality with static HTML, CSS, and JavaScript integration.

---

## 🌐 Live Demo

👉 Click here to view the application:
🔗 https://springbootdemo-r1br.onrender.com/

---

## 📌 Features

* Simple Spring Boot setup
* Static HTML page rendering
* Basic JavaScript interactions
* Dockerized application
* Deployed on Render

---

## 🛠️ Tech Stack

* Java (Spring Boot)
* HTML, CSS, JavaScript
* Maven
* Docker
* Render (Deployment)

---

## 🚀 How to Run Locally

1. Clone the repository:

```
git clone https://github.com/Anasurya-Reddy/SpringBootDemo.git
```

2. Navigate to the project folder:

```
cd SpringBootDemo
```

3. Run the application:

```
mvn spring-boot:run
```

4. Open in browser:

```
http://localhost:8080
```
